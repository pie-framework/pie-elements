### BREAKING CHANGES

* **model:** Model has changed.

FROM:

```javascript

    {
      model: {
        config: {
          units: 'metric',
          label: 'm',
          length: 20,
          pixelsPerUnit: 30,
          ticks: 10
        }
      }
    }

```

TO:

```javascript
   {
      measure: 'metric',
      label: 'm',
      units: 10,
      width: 500,
      imperialTicks: undefined
    }
```
