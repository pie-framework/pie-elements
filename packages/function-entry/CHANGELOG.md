* **model:** move away from legacy model, use latest feedback model. ([317a9c3](https://github.com/pie-framework/pie-elements/commit/317a9c3))


### BREAKING CHANGES

* **model:** The model has moved from this:

```javascript
{
id: '1',
element: 'function-entry',
weight: 1,
incorrectFeedback: {
type: 'default',
},
correctResponse: {
equation: '3x+2',
feedback: {
type: 'default',
},
},
model: {
showFormattingHelp: true
}
}

```

To this:

```javascript
{
id: '1',
element: 'function-entry',
weight: 1,
showFormattingHelp: true,
equation: '3x+2',
feedback: {
correct: {
type: 'default',
default: 'Correct'
},
incorrect: {
type: 'default',
default: 'Incorrect'
}
}
}

```
